Portfolio Projects

Global Population Analysis in Python

The world population is growing fast, but not exponentially as many think. In this project, I used Python to analyse the global population data from 1950 to 2023 and examine the factors that affect its growth rate. I used Python libraries pandas, numpy, matplotlib, and seaborn to clean, manipulate, visualize, and analyse the data. I discovered that the population growth rate is linear, not exponential, with a high R^2 of 0.9967. I also explored the correlation between the population growth rate and other variables such as birth rate, poverty rate, and contraception prevalence. Please click the image to go to my GitHub repository for more information.

World Stock Prices Dashboard

For this project, I used the World Stock Prices dataset from Kaggle to create a dashboard that can provide quick and meaningful insights.
